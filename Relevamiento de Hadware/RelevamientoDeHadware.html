Que es un relevamiento de hardware

Es el proceso de identificar y registrar los componentes fisicos de una computadora. Su objetivo es conocer el estado del equipo para mantenimiento y control.

Componentes a relevar

  • CPU
  • Memoria RAM
  • Disco rigido o SSD
  • Placa madre
  • Fuente
  • Placa de video
  • Perifericos

Pasos para realizarlo

  1. Identificar el equipo
  2. Revisar componentes
  3. Anotar caracteristicas
  4. Registrar estado
  5. Guardar informacion
